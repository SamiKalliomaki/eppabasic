Max
===

```eppabasic
Function Max(a As Number, b As Number) As Number
Function Max(a As Integer, b As Integer) As Integer
```

Returns the bigger of the two numbers a and b.

Example
---------
```eppabasic
Print Max(5, 7)       ' 7
Print Max(-2, -3)     ' -2
Print Max(4, 4)       ' 4
```
