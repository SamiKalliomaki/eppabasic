Abs
===

```eppabasic
Function Abs(luku As Number) As Number
Function Abs(luku As Integer) As Integer
```

Palauttaa parametrin `luku` itseisarvon eli luvun etäisyydden nollasta.
Toisin sanoen palauttaa luvun ilman etumerkkiä.

Esimerkki
---------
```eppabasic
Print Abs(10)     ' 10
Print Abs(-10)    ' 10
Print Abs(-3.14)  ' 3.14
```
